01 — Overview

SDKとは何か

"Extensions are optional add-on tools that run alongside Live and can be accessed with a right-click from anywhere in your Set." — Ableton 公式ブログ（2026-06-02）

Ableton Live Extensions SDK は、2026年6月2日にパブリックベータとして公開された 無償・オープンな JavaScript ツールキットです。 これにより、ユーザーは Live 12 Suite 内で動作するカスタムツール（Extensions）を JavaScript / TypeScript で自作し、セッション内のトラック・クリップ・パラメータ・オートメーションなどを 自由に読み書きできるようになります。

Extensions は Live のコンテキストメニュー（右クリック） から起動し、 関連するセット構造に直接アクセスします。 Max for Live が「デバイスチェーン上のリアルタイム処理」に強いのに対し、 Extensions は「セット全体の構造操作・ワークフロー自動化・外部サービス連携」を担う 別レイヤーの拡張機構です。

Ableton は「AI コーディングアシスタントが SDK の基礎技術（Web 標準）を熟知しているため、 コーディング経験がなくてもアイデアを明確に説明できれば Extension が作れる」と明言しており、 AI × 音楽制作 の最前線に位置する取り組みです。

🔓

完全オープン・無償

SDKは無料。Live 12 Suite のライセンスがあればベータ版で今すぐ使える。

🖱️

右クリックから呼び出す

セット内どこからでもコンテキストメニュー経由で Extension を起動。自然な UX。

🤖

AI で作れる

Web 標準ベースなので Claude・Copilot 等の AI アシストが効き、非エンジニアでも参入可能。

03 — Capabilities

できること・できないこと

✅ できること

  • ✓セット構造の読み取り・書き換え（トラック・クリップ・パラメータ）
  • ✓MIDI ノートの生成・変換・編集
  • ✓トラック名・カラーのバッチ操作
  • ✓オートメーションの読み書き
  • ✓外部サービス（HTTP / API）への接続
  • ✓カスタム UI（WebView・モーダル）の構築
  • ✓npm パッケージの活用（任意のJSライブラリ）
  • ✓セッション分析・可視化・整理の自動化
  • ✓テンポ・タイム署名の操作
  • ✓アイデア生成・ランダム変換ツール

✗ できないこと

  • ✗リアルタイムオーディオ処理（エフェクト・合成・フィルター）
  • ✗リアルタイム MIDI モニタリング（演奏中の変換）
  • ✗シンセ・サンプラー・オーディオエフェクトの開発
  • ✗再生・録音中のバックグラウンド処理
  • ✗サンプル精度・イベント精度のタイミング処理
  • ✗デバイスチェーン上への組み込み
  • ✗Live Standard / Intro / Lite での使用
Extensions は「リアルタイム処理」ではなく「構造・データ・ワークフロー」の層を担う。 リアルタイム処理が必要なら Max for Live、構造操作・自動化なら Extensions — という棲み分け。
04 — Comparison

Extensions vs Max for Live

観点 Extensions SDK Max for Live
開発言語 JavaScript / TypeScript（Web標準） Max パッチ（ビジュアルプログラミング）
リアルタイム処理 ✗ 不可 ✓ 可（サンプル精度）
セット全体の構造操作 ✓ 強力な API △ 限定的
コンテキストメニュー統合 ✓ ネイティブ ✗ なし
カスタム UI（WebView） ✓ HTML/CSS/JS △ Max UI のみ
AI コーディングアシスト相性 ✓ 非常に良い（Web標準） △ やや限定的
外部サービス連携 ✓ npm フルアクセス △ 可能だが複雑
コスト 無料（SDK のみ） Cycling '74 ライセンス（Suite に同梱）
学習コスト Web 開発知識があれば低い Max 独自の概念習得が必要
デバイスチェーンへの組み込み ✗ 不可 ✓ 可

活用アイデア集（オリジナル提案）

Extensions SDK でできることを最大化する、思いつく限りのアイデア。難易度バーは目安（実装者のJS習熟度による）。

🗂️ Smart Session Organizer
実用 初級
右クリックひとつで、トラック種別（Audio/MIDI/Return/Master）を自動判定し、カラーコーディングと名前フォーマット統一を実行。「drums_」「bass_」「synth_」などのプレフィックスを自動付与するルールも設定可能。
難易度
📦 Session Snapshot & Diff
実用 中級
セットの「スナップショット（トラック構成・デバイス・パラメータ値）」を JSON として保存し、後から diff で変化を確認。セッション versioning を Live ネイティブに実現。
活用場面: 大幅に試した後で「さっきの状態に戻したい」ときや、コラボ作業での変更確認。
難易度
🎛️ Batch Parameter Reset / Copy
実用 初級
選択した複数トラックの特定デバイスパラメータを一括リセット、または別トラックのパラメータをコピペ。例: 全 EQ Eight の High gain を同じ値に揃える。

実用 初級
