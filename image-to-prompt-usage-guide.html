/image-to-prompt

添付画像を観察して、限りなく忠実に再現できる画像生成プロンプトを
Stable Diffusion タグ形式 と 自然言語形式 の両方で自動生成するスキル。

1これは何をするスキルか

画像を1枚渡すだけで、以下をまとめて出力する：

  • 画像の画風・被写体・構図・光・色調などを10軸で徹底観察
  • その画風に合う 推奨モデル（checkpoint）と LoRA を推定して提案
  • SD タグ形式（Positive / Negative / 推奨パラメータ）
  • 自然言語形式（Flux・Midjourney・DALL-E・Imagen 向け）
  • 「髪色を赤に」等の指示に対する差分編集（他の部分は一切変えない）
置き場所： グローバル配置（~/.claude/skills/image-to-prompt/）なので、 Claude Code・Claude.ai を問わず、どのプロジェクトからでも呼び出せる。

2基本の使い方

  1. チャットに画像を添付する
  2. 「このプロンプト作って」「同じ画像を再現したい」などと頼む（スキル名を言わなくてもトリガーされる）
  3. SD タグ形式・自然言語形式・推奨モデル/LoRA・再現の限界が返る
  4. 結果は自動的にファイル保存される（差分編集の前提になるので必須）

3差分編集モード（属性だけ変える）

一度プロンプトを生成した後、続けて一部だけ変えたいときに使う。指定した箇所以外は一切変更されない。

You
髪色を赤にして
Claude
blonde hair, pastel rainbow hair → red hair, pastel red gradient hair のみ変更。 服装・構図・背景・光・画風タグは元のまま。変更箇所を太字でマークして同じファイルに v2 として追記。
指示の例変更されるもの
「髪色を赤にして」髪色タグのみ
「背景を夜にして」背景・光の記述（昼要素と矛盾する部分のみ連動）
「服をスーツに変えて」服装タグのみ

連動変更が発生した場合は、何を・なぜ連動させたかを明記する。

4モデル・LoRA の推定

画風判定から、Civitai エコシステムの定番 checkpoint 系統を推定して提案する。

画風推奨モデル系統方言
現代アニメ・イラストIllustrious-XL / NoobAI-XLDanbooru タグ
デフォルメ・カートゥーンPony Diffusion V6 XLscore タグ
写実ポートレートJuggernaut XL / RealVisXL自然英語句
高精細・文字入り・複雑構図Flux.1 dev自然文（ネガティブなし）
推定は仮説： モデル名・LoRA名は執筆時点の定番から推定しているだけなので、 毎回「Civitai で〇〇と検索して確認」という検証手順とセットで返す。存在が不確かな名前を断定しない。

5NSFW への対応

成人向け画像も SFW と同じ精度・同じ手順で解析する。露出や性的要素も婉曲・省略せず正確なタグで記述する（曖昧化すると再現度が落ちるため）。

絶対的除外（解析自体を拒否）：
  1. 未成年、または未成年に見える被写体・キャラクターの性的描写
  2. 実在人物の性的・中傷的コンテンツ
  3. 非合意的行為を現実の出来事として美化・指南する内容

このスキルは指示書であり強制力を持たない。実行するAIモデル自身の安全ポリシーが常に優先される。

7再現度をさらに上げたいとき

プロンプトだけでは詰め切れない要素は、出力の「再現の限界」欄で個別に提案される：

  • 特定人物・キャラの顔 → character LoRA の学習、または IP-Adapter / InstantID / PuLID
  • 構図・ポーズの完全一致 → ControlNet（OpenPose / Depth / Canny）
  • 画風の完全一致 → style LoRA（なければ元画像で自作学習）
  • 微調整 → img2img（denoise 0.3〜0.5）
